# Notes API

A simple RESTful API for managing notes using ASP.NET Core Web API.

## Features

- Register
- Login
- JWT Authentication
- CRUD Operations
- Search Notes
- Pagination
- Each user can manage only their own notes

## Technologies

- ASP.NET Core Web API
- Entity Framework Core
- SQL Server
- ASP.NET Identity
- JWT
